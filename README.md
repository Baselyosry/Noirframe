# NoirFrame

NoirFrame is an AI-guided framework for turning a project brief into a
traceable engineering knowledge base.

## Download

Download `NoirFrame.zip` for the public framework package. It contains the
Framework Core, templates, generator prompts, review prompts, Agent Layer, and
validation template. Internal design notes, reviews, and dogfood evidence are
not included.

## Start Here

Open [`NoirFrame/README.md`](NoirFrame/README.md) for the complete bootstrap
guide. You can use a project brief in PDF, DOCX, Markdown, plain text, or any
other format your AI agent can read.
